# 💰 Smart Personal Finance Tracker

## 📌 Project Overview
This project analyzes personal expenses to track spending behavior, generate insights, and visualize financial patterns.

---

## 🎯 Objectives
- Monitor daily expenses  
- Analyze spending by category  
- Identify highest and lowest spending days  
- Predict future expenses  
- Detect trends in spending behavior  

---

## 📂 Dataset
The dataset contains:

- `id` → Transaction ID  
- `name` → Person name  
- `amount` → Expense amount  
- `category` → Expense category  
- `date` → Transaction date  

---

## 🧹 Data Cleaning
- Converted `date` to datetime format  
- Removed missing values  
- Removed duplicate records  

---

## 📊 Data Analysis
- 💰 Total Spending  
- 📉 Average Spending  
- 📅 Daily Expenses  
- 📊 Category-wise Spending  
- 📆 Strongest & Weakest Day  
- 🔮 Expense Forecasting  
- 💸 Budget Comparison  
- 📈 Trend Analysis  

---

## 📈 Visualizations
- 📊 Bar Chart — Expenses by Category  
- 📈 Line Plot — Expenses Over Time  
- 🥧 Pie Chart — Distribution by Category  
- 🔥 Heatmap — Correlation  
- 📉 KDE Plot — Distribution of Expenses  
- 📍 Rug Plot — Data Distribution Points  
- 🎻 Violin Plot — Distribution by Category  
- 🐝 Swarm Plot — Detailed Distribution  
- 📦 Box Plot — Outliers & Spread  
- 🔗 Pair Plot — Feature Relationships  

---

## 🔍 Key Insights
- Food is the highest spending category  
- Spending shows a fluctuating but increasing trend  
- Certain days have significantly higher expenses  
- Forecast indicates potential budget overflow
